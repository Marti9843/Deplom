Дитячий басейн

Для дітей від 0 до 14 років

Безпечний басейн з підігрівом води та спеціальним покриттям. Глибина 0.85м, температура води +32°C. Ідеальне місце для перших кроків у навчанні плаванню.

Розмір: 6×10 м

Спортивний басейн

25-метровий професійний

8 доріжок для плавання, глибина від 1.8 до 2.9 м. Система протитечії, стартові тумби. Ідеально підходить для тренувань та змагань.

Температура води: +28°C

Навчання дітей плаванню

Для дітей 4-12 років

Професійне навчання плаванню для дітей різних вікових груп. Використання ігрових методик, поетапне навчання основним стилям плавання.

Тривалість: 45 хв

Аквааеробіка

Для дорослих

Ефективні тренування у воді, що поєднують аеробні вправи з силовими. Ідеально для схуднення, підвищення витривалості та зміцнення м'язів.

Тривалість: 60 хв

Спортивне плавання

Для дорослих та підлітків

Тренування з вивчення техніки всіх спортивних стилів плавання. Підвищення швидкості та витривалості. Індивідуальний підхід до кожного.

Тривалість: 60-90 хв

Раннє плавання

Для немовлят 0-3 роки

Спеціальні заняття для найменших разом з батьками. Сприяє фізичному розвитку, зміцненню імунітету та адаптації до водного середовища.

Тривалість: 30 хв

Аква-йога

Для всіх вікових груп

Поєднання класичних асан йоги з особливостями води. Розслабляє, зміцнює м'язи, покращує гнучкість та баланс. Ідеально для реабілітації.

Тривалість: 60 хв
